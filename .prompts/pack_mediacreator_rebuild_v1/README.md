
# pack_mediacreator_rebuild_v1

This pack is designed to be unzipped into `/opt/MediaCreator` and executed by Codex in a brand-new empty folder.

## What is inside

- `SOURCES.md` — master reference list of official docs and external source material
- `STACK_DECISIONS.md` — the chosen stack and architecture
- `MASTER_RUN.md` — the overall mission and execution contract
- `RUN_ORDER.md` — exact phase order
- `PHASE_SUMMARY.md` — one-line summary of all phases
- `REPORT_TEMPLATE.md` — reporting contract
- `phases/` — build and verify documents for every phase

## Important

This pack assumes Codex is working in a brand-new folder and should build the app from scratch.
Do not mix this pack with older AutoCharacter prompt packs.
